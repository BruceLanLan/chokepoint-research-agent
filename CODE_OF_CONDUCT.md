# Code of Conduct

## Our pledge

Be respectful. This is an open research-tooling project.

## Unacceptable behavior

- Harassment, hate speech, or personal attacks
- Publishing others’ private information
- Spam, phishing, or malicious content
- Using project spaces to solicit market manipulation or “pump” schemes
- Presenting project outputs as guaranteed investment advice

## Enforcement

Maintainers may remove content or block contributors who violate this code.

Report issues via GitHub: [@BruceLanLan](https://github.com/BruceLanLan)
