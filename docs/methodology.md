## The question

中文：在这个系统里，谁是那个沉默的、不可替代的物理开关？

## Bottom-up vs top-down

| | Top-down (common sell-side) | Bottom-up chokepoint |
|--|-----------------------------|----------------------|
| Start | NVDA / hyperscaler Capex | Physical cluster / product |
| Ask | Will EPS beat next quarter? | Which node freezes the chain if cut? |
| Coverage bias | Large-cap, liquid | Often mid/small, multi-market |
| Validation | Consensus models | Physical barriers + red-team |

## Scorecard dimensions

Each candidate node can be scored 1–5 on:

1. **Irreplaceability** — years to re-qualify alternatives  
2. **Supply concentration** — 1–3 vendors / single geography  
3. **Downstream leverage** — impact if disrupted  
4. **Institutional vacuum** — under-covered by street research  
5. **Commercial inflection** — moving from PPT to volume  

## Red-team before publish

Always attack:

- Single-path technology risk (e.g. CPO vs advanced copper)  
- Valuation that prices perfection  
- Liquidity traps in micro-caps  
- Unverifiable narrative / persona claims
