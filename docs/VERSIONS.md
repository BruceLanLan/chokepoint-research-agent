# Major versions (0.3 → 0.7)

Implemented overnight as five sequential product versions.

| Version | Theme | Highlights |
|---------|-------|------------|
| **0.3** | China data + eval + stream UI | `get_cn_stock_quote`, CN news helper, golden eval harness, SSE UI |
| **0.4** | Compare + memory | `compare` mode, session store, multi-turn context |
| **0.5** | Resilience + export | retries, cost estimates, MD/JSON/HTML export |
| **0.6** | Deploy + maps + bilingual | EdgeOne guide, HBM/pluggable maps, bilingual one-pager |
| **0.7** | Release polish | docs, tests, version alignment |

Each version kept the standing rules:

1. Framework over tickers  
2. Tools own facts when possible  
3. Red-team / kill criteria  
4. Explicit non-advice disclaimer
