# 产品路线图 — Chokepoint Research Agent

> 目标：做成**专业投研工作台**，而不是聊天玩具。  
> 对标：Bloomberg/FactSet 的研究流程、Koyfin/TIKR 的基本面、AlphaSense 的检索、Perplexity Finance 的综合、TradingView 的监控。

**English → [../ROADMAP.md](../ROADMAP.md)**

---

## 1. 竞品差距

| 能力 | 专业平台 | 我们（v0.7） | 差距 |
|------|----------|--------------|------|
| 多源行情/财报 | 终端级实时 | 尽力公开接口 | 高 |
| 结构化研究流程 | 分析师模板 | 多 Agent + 模式 | 中 → 收敛中 |
| 自选股 / 覆盖名单 | 标配 | 无 → **v0.8** | 收敛中 |
| 论点台账 + 证伪条件 | 内部 wiki | 仅 Prompt → **v0.8** | 收敛中 |
| 研报库检索 | 企业文档库 | 文件夹 → **v0.8** | 收敛中 |
| 工作台仪表盘 | 完整 BI | 单页 → **v0.9** | 收敛中 |
| 定时简报 | 提醒/摘要 | 无 → **v0.10** | 收敛中 |
| 多用户权限 | 企业标配 | 可选 API Key | 以后 |
| 组合盈亏 / 下单 | 核心产品 | 合规上不做 | 永不做「投顾」 |

**定位一句话：**  
开源、框架优先的投研 Agent，把「卡脖子 / 供应链逆向工程」工业化：覆盖名单、论点台账、研究模板、研报目录、定时简报。

---

## 2. 原则（不可妥协）

1. **研究 ≠ 投资建议**  
2. **事实尽量走工具**  
3. **发布前红蓝对抗**  
4. **可审计**（文件 + 元数据）  
5. **每次用户可见更新都有中英文档**
