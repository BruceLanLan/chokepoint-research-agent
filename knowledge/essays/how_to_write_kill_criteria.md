# How To Write Kill Criteria

Educational essay for Chokepoint research process.

## Thesis

Research discipline matters more than narrative heat. Document system boundaries,
physical nodes, and falsifiers before discussing equities as proxies.

## Practice

1. Write the system in one sentence.
2. List nodes that fail open-source substitution tests.
3. Attach dated primary sources.
4. Draft kill criteria that would change your mind.
5. Run checklist / grade-memo before sharing.

## Anti-patterns

- Numbers without dates or sources
- Equating ticker stories with physical bottlenecks
- Skipping red-team because the story "feels" right
